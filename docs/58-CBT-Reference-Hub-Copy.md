# CBT Reference Hub Copy

## Copy Approach
The page copy was written as a simplified restatement of the trainer material. The writing standard used was:

- beginner-friendly
- practical
- short enough for scanning
- explicit about use in the training product
- faithful to the source material

## Core Reference Copy Blocks

### What Is CBT
Explains CBT as:
- structured
- practical
- collaborative
- educational
- based on the link between thoughts, emotions, behavior, and meaning

### Principles
Condensed into practical statements covering:
- progressive assessment
- collaboration
- clear goals
- present focus
- educational nature
- structured sessions
- time-bounded work
- multiple techniques

### Session Structure
Rendered as a fixed checklist:
1. mood/status check
2. agenda
3. review homework
4. main work
5. summary
6. new homework

### 5-Session Map
Each session uses the same scanning frame:
- purpose
- main tools
- expected homework
- expected output

### Tools Library
Each tool card uses the same compact structure:
- what it is
- when to use it
- why to use it
- short example
- common mistake

### Thinking Levels
Each level includes:
- short definition
- how it differs from the others
- one practical example

### Distortions Reference
Each distortion includes:
- simple definition
- short example
- how it usually appears in real thinking

### Emotions Reference
Built as core emotion groups with sub-emotions to support worksheet use and emotion labeling.

### Worksheets Reference
Explains what each worksheet is for and what fields it usually includes.

### Goals and Treatment Plan
Explains immediate, medium-term, and long-term goals and links treatment planning back to formulation and problem list.

### Glossary
Plain-language explanations only. No academic expansion beyond the trainer material.

## Advanced Reference Copy Blocks
Advanced sections were rewritten as compact training notes, not full theory chapters. The intention is to help when a trainee or facilitator needs more depth without displacing the beginner-facing flow.
