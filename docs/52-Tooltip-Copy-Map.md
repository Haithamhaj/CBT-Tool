# Tooltip Copy Map

## Practice Setup

### Session goal

- English: What should be achieved by the end of this practice? Example: identify one automatic thought linked to the anxiety spike.
- Arabic: ما الذي يجب تحقيقه بنهاية هذه الممارسة؟ مثال: تحديد فكرة تلقائية واحدة مرتبطة بارتفاع القلق.

### Tool rationale

- English: Explain why this tool fits this stage and case. Example: a thought record fits when the goal is to examine a specific automatic thought.
- Arabic: اشرح سبب ملاءمة الأداة لهذه المرحلة وهذه الحالة. مثال: يناسب سجل الأفكار فحص فكرة تلقائية محددة.

## Session Step

### Step input

- English: Write the trainee’s actual CBT response, not notes to yourself. Keep it concrete and session-focused.
- Arabic: اكتب الاستجابة العلاجية الفعلية للمتدرب، وليس ملاحظات لنفسك. اجعلها عملية ومرتبطة بالجلسة.

### Session summary

- English: End with a brief recap of the main point learned or clarified. Example: the client linked anxiety to a prediction about embarrassment.
- Arabic: اختم بخلاصة قصيرة للنقطة الأساسية التي اتضحت. مثال: ربط العميل القلق بتوقع الإحراج.

### Homework

- English: Assign one small task that follows directly from the session. Example: complete one thought record after the next difficult meeting.
- Arabic: أعط مهمة واحدة صغيرة تنبثق مباشرة من الجلسة. مثال: إكمال سجل أفكار واحد بعد الاجتماع الصعب القادم.

## Review

### Official Review

- English: This is the authoritative result from deterministic validation and scoring. Treat it as the final session outcome.
- Arabic: هذه هي النتيجة المرجعية الناتجة عن التحقق الحتمي واحتساب الدرجات. اعتبرها النتيجة النهائية للجلسة.

### Advisory Guidance

- English: This section is supportive guidance only. It can explain patterns but cannot change the official result.
- Arabic: هذا القسم داعم فقط. يمكنه شرح الأنماط لكنه لا يغيّر النتيجة الرسمية.

### Next action

- English: This is the main action the trainee should take next based on the official review.
- Arabic: هذا هو الإجراء الأساسي الذي ينبغي على المتدرب القيام به بعد هذه المراجعة الرسمية.

### Next focus

- English: This is an advisory practice focus suggested by the synthesizer. It should support, not replace, the official next action.
- Arabic: هذا تركيز تدريبي استشاري تقترحه خلاصة الجلسة. يجب أن يدعم الإجراء التالي الرسمي لا أن يستبدله.

## Copy Rules

- keep each tooltip under roughly two short sentences
- give one example only when it removes ambiguity
- avoid theory language
- explain expected output, not system internals
