# Reference Hub Information Architecture

## Page Role
The page is designed as a practical reference hub that users can scan in short bursts during training.

## Top-Level Structure

### 0. Quick Start + Tabs
Purpose:
- stop the page from behaving like one long article
- route the user to the right operational section quickly
- make the page usable before, during, and after training sessions

### 1. Hero / Orientation
Purpose:
- explain what this page is for
- position it as the study/reference map for the trainer material

### 2. Core Reference
Priority:
- primary
- always visible
- first stop for beginners when its tab is active

Order:
1. what CBT is
2. core principles
3. fixed session structure
4. 5-session exploration map
5. tools library
6. levels of thinking
7. cognitive distortions
8. emotion reference
9. worksheets/templates
10. goals and treatment planning
11. glossary

Reason for this order:
- starts with orientation
- then gives the session frame
- then shows the early training path
- then gives tools and conceptual references
- ends with supporting reference material

### 3. Advanced Reference
Priority:
- secondary
- collapsed
- for deeper formulation and assessment only

Order:
1. deeper formulation breakdown
2. triggering and predisposing factors
3. severity and assessment reference
4. working hypotheses
5. assessment/diagnostic reference

## Representation Choices

### Tabs
Used for:
- major section switching
- faster scanning than long-scroll-only behavior

### Quick Start cards
Used for:
- "I am new"
- "before session"
- "during session"
- "after session"
- "deeper understanding"

### Diagrams
Used for:
- CBT model
- session flow
- five-session progression

### Checklists
Used for:
- session structure
- before / during / end-of-session action support
- worksheet inclusions
- advanced bullets

### Cards
Used for:
- principles
- tools
- thinking levels
- distortions
- goals

### Session Map Cards
Used for:
- the 5-session exploration path

### Emotion Cards with Chips
Used for:
- core emotions plus sub-emotions

### Collapsible Blocks
Used for:
- advanced reference

## Language Architecture
- one content tree for English
- one content tree for Arabic
- no mixed-language rendering on the page
- same information architecture in both languages
